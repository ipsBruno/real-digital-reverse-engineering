# Real Digital CBDC Engenharia Reversa 🕵️‍♂️💻

Este projeto é uma tentativa de realizar engenharia reversa nos contratos inteligentes da Real Digital Central Bank Digital Currency (CBDC). Como o código-fonte da Real Digital não foi disponibilizado publicamente, apenas os ABIs foram fornecidos. Portanto, decidi realizar a engenharia reversa e inferir a funcionalidade de algumas das funções.

Por favor, note que este projeto é baseado em esforços de engenharia reversa e pode não capturar completamente a implementação real ou o comportamento pretendido da Real Digital CBDC. Use este código apenas para referência e fins educacionais.

## Aviso Legal ⚠️

Os contratos da Real Digital CBDC são propriedade do Banco Central e de sua equipe de desenvolvimento. Este projeto não tem afiliação ou endosso do Banco Central. Use este projeto por sua própria conta e risco.

## Recursos ✨

- Implementação de engenharia reversa de funções selecionadas da Real Digital CBDC
- Recurso educacional para entender estruturas e comportamentos de contratos CBDC

## Começando 🚀

Para começar com o projeto de Engenharia Reversa da Real Digital CBDC, siga estes passos:

1. Clone este repositório
2. Instale as dependências: `npm install`
3. Execute os testes do Hardhat: `npx hardhat test`
4. Revise o código fornecido e a documentação para entender a funcionalidade inferida dos contratos da Real Digital CBDC.
5. Personalize e experimente o código de acordo com seus propósitos de aprendizado ou pesquisa.
6. Compartilhe suas descobertas e insights com a comunidade, abrindo problemas ou enviando solicitações de pull no repositório do GitHub.
